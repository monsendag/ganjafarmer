HTML5 based 2d game

This game was made as my contribution to the Opera Software international internship competition. Which i won :)

The objective is to defend your crops from the attacking forces. They are attacking from above, and in great numbers. For each 30 kills, their rate of attack will increase. When your 26 crops are burned out, the game is over.

A number of technologies are used

  • CSS3 Selectors
  • HTML5 Canvas element
  • HTML5 Audio
  • HTML5 Localstorage

The game has been tested, and should work well in the latest version of the following browsers.
